# python_practice_assignment_5

# Assignment 5 - Module 6: Data Structures and Strings in Python

This repository contains solutions for **Assignment 5** of **Module 6: Data Structures and Strings in Python**.
The assignment focuses on using **dictionaries** and **list slicing** in Python.

---

## 📌 Tasks Included

### Task 1: Dictionary of Student Marks

**Description:**

* Creates a dictionary with student names as keys and marks as values
* Takes user input to search for a student's marks
* Retrieves and displays the corresponding marks
* Handles cases where the student is not found

**File:**
`task_1/student_marks.py`

---

### Task 2: List Slicing and Reversal

**Description:**

* Creates a list of numbers from 1 to 10
* Extracts the first five elements using slicing
* Reverses the extracted elements
* Displays both original and processed lists

**File:**
`task_2/list_slicing.py`
